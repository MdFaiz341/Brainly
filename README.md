# 📖 Overview
 - Second Brain App is a personal content management system that helps users save, organize, and    retrieve useful content from platforms like YouTube and Twitter.

 - Instead of losing valuable resources, users can store them in one place and access them anytime.

# ✨ Features
 - 🔐 Authentication System
    - Secure Signup & Signin
 - 📌 Save Content
    - Store YouTube videos
    - Save Twitter posts
 - 🔍 Smart Search
    - Quickly find saved content
 - 🎯 Platform Filters
    - Filter by YouTube
    - Filter by Twitter
 - ⚙️ User Settings
    - Change password
    - Delete account
 - 🌗 Theme Toggle
    - Dark Mode / Light Mode
 - 🖥️ Modern UI
    - Responsive and clean design

# Clone the repo

## Setup Backend
 - cd server
 - npm install
 - npm run dev

## Setup Frontend
 - cd client
 - npm install
 - npm run build
 - npm run dev

# Environment Variables
 - Create a .env file in the server folder:
    - PORT=5000
    - MONGO_URI=your_mongodb_connection
    - JWT_SECRET=your_secret_key

# Usage
 - Signup or Signin
 - Paste a YouTube or Twitter link
 - Save the content
 - Use search & filters to find it later
